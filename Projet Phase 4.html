Aperçu¶

Le projet consiste concevoir un systme de recommandation de films capable de proposer chaque utilisateur les films quil est le plus susceptible dapprcier. Pour ce faire, nous avons utilis lensemble de donnes MovieLens 100k, fourni par le laboratoire GroupLens de lUniversit du Minnesota.

  • Lensemble de donnes contient 100 000 valuations provenant de plusieurs milliers dutilisateurs et de milliers de films.

  • Chaque valuation est note sur une chelle de 0.5 5 toiles.

  • Le projet utilise une approche de filtrage collaboratif, complte par un filtrage bas sur le contenu pour crer un modle hybride capable de grer le problme du dmarrage froid.

Objectif¶

Objectif principal :

  • Fournir chaque utilisateur une liste personnalise de 5 films recommands bass sur ses prfrences passes et les comportements dautres utilisateurs.

Objectifs secondaires :

  • valuer la performance du modle laide de mtriques standards telles que RMSE et MAE.
  • Explorer la combinaison de mthodes collaboratives et bases sur le contenu pour amliorer la prcision et la ro

Problmatique mtier : Comment maximiser la satisfaction des utilisateurs en anticipant les films quils aimeront, tout en grant les nouveaux films et nouveaux utilisateurs sans donnes historiques ?

Résumé¶

Fichiers chargs :

  • movies.csv : contient les films avec movieId, title et genres (plusieurs genres possibles, spars par |).
  • ratings.csv : contient les notes des utilisateurs (userId, movieId, rating, timestamp).
  • links.csv : relie chaque film ses identifiants IMDb et TMDb, utile pour enrichir les donnes.
  • tags.csv : mots-cls attribus par les utilisateurs pour dcrire les films, pouvant amliorer le filtrage bas sur le contenu.

Structure et observations :

  • movies : 3 683 films, genres multiples utile pour analyser les prfrences par genre.
  • ratings : beaucoup de lignes capture les interactions utilisateur-film, indispensable pour le filtrage collaboratif.
  • tags : informations qualitatives possibilit dajouter des recommandations bases sur le contenu.

Insight cl :

  • Les donnes sont compltes et cohrentes, permettant de construire un systme de recommandation hybride, combinant les prfrences explicites des utilisateurs (notes) et les caractristiques des films (genres et tags).

from collections import defaultdict

def get_top_n(predictions, n=5):
    """Retourne les n films les mieux nots pour chaque utilisateur"""
    top_n = defaultdict(list)
    for uid, iid, true_r, est, _ in predictions:
        top_n[uid].append((iid, est))
    for uid, user_ratings in top_n.items():
        user_ratings.sort(key=lambda x: x[1], reverse=True)
        top_n[uid] = user_ratings[:n]
    return top_n

def content_based_recommendations(user_rated_movie_ids, top_n=5):
    """Recommande des films similaires  ceux que l'utilisateur a dj aims"""
    sim_scores = {}
    for movie_id in user_rated_movie_ids:
        idx = movie_indices[movie_id]
        sim = list(enumerate(cosine_sim[idx]))
        for i, score in sim:
            movie_i_id = movies.iloc[i]['movieId']
            if movie_i_id not in user_rated_movie_ids:  # Ignorer films dj vus
                if movie_i_id in sim_scores:
                    sim_scores[movie_i_id] += score
                else:
                    sim_scores[movie_i_id] = score
    # Trier les scores et prendre top_n
    top_movies = sorted(sim_scores.items(), key=lambda x: x[1], reverse=True)[:top_n]
    recommended_titles = [movies[movies['movieId']==mid]['title'].values[0] for mid, _ in top_movies]
    return recommended_titles

# Recommandations hybrides
def hybrid_recommendations(user_id, top_n=5):
    # 1. Films dj vus
    user_movies = ratings[ratings['userId']==user_id]['movieId'].tolist()
    
    # Recommandations filtrage collaboratif
    movies_to_predict = movies[~movies['movieId'].isin(user_movies)]
    cf_predictions = [algo.predict(user_id, mid) for mid in movies_to_predict['movieId']]
    top_cf = sorted(cf_predictions, key=lambda x: x.est, reverse=True)[:top_n//2]
    cf_titles = [movies[movies['movieId']==pred.iid]['title'].values[0] for pred in top_cf]
    
    # Recommandations contenu
    content_titles = content_based_recommendations(user_movies, top_n=top_n//2)
    
    # Fusion
    hybrid_titles = cf_titles + content_titles
    return hybrid_titles

# Exemple : top-5 recommandations hybrides
hybrid_recs = hybrid_recommendations(user_id, top_n=5)
print("Recommandations hybrides :")

    print(f"{i}. {title}")
